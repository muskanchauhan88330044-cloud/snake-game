# 🐍 Hand Gesture Snake Game 🎮

Control the classic Snake Game using your **hand gestures** instead of a keyboard!
This project uses **Computer Vision** to detect your hand movements and turn them into game controls.

---

## 🚀 Features

* ✋ Control snake with hand gestures
* 🎥 Real-time hand tracking using webcam
* 🧠 Powered by Computer Vision
* 🐍 Classic snake gameplay
* ⚡ Smooth and interactive experience

---

## 🛠️ Tech Stack

* Python 🐍
* OpenCV
* MediaPipe
* Pygame

---

## 📦 Installation

### 1. Clone the repository

```bash
git clone https://github.com/yamanlohan78-web/Snake-Game.git
cd Snake-Game
```

### 2. Create virtual environment

```bash
python -m venv venv
```

### 3. Activate virtual environment

**Windows:**

```bash
venv\Scripts\activate
```

### 4. Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ How to Run

```bash
python main.py
```

Make sure your webcam is enabled 📷

---

## 🎮 Controls

* Move your hand **left/right/up/down** to control the snake
* Try to eat food and grow longer
* Avoid colliding with yourself
